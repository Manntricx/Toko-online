# Toko Online Sederhana

Proyek ini adalah halaman web HTML sederhana untuk toko online. Ini mencakup header, navigasi, daftar produk, dan footer.

## Cara Menggunakan

1. Buka file `index.html` di browser web Anda.
2. Jelajahi bagian Home, Produk, Tentang, dan Kontak melalui navigasi.
3. Produk ditampilkan dengan gambar placeholder, nama, harga, dan tombol "Tambah ke Keranjang" (statis, tidak berfungsi).

## Fitur

- Desain responsif dengan CSS.
- Navigasi sederhana.
- Daftar produk statis.

## Troubleshooting

- Jika gambar tidak muncul, pastikan koneksi internet aktif karena menggunakan placeholder dari via.placeholder.com.
- Untuk menambahkan fungsionalitas dinamis (seperti keranjang belanja), tambahkan JavaScript atau gunakan framework seperti React.

## Lisensi

Proyek ini untuk tujuan edukasi.
